# 👮 WARDEN

> **Real-Time Security & Vulnerability Guardian**

WARDEN is the security specialist of the SKRYMIR suite. It monitors for sensitive data leaks (API keys, secrets), outdated dependencies with known vulnerabilities, and potential security flaws in your code logic.

---

## 🚀 Key Responsibilities

- **Secret Detection:** Scans for hardcoded credentials in real-time.
- **Vulnerability Scanning:** Checks project dependencies against security databases.
- **Compliance Guard:** Ensures the project follows basic security hardening rules.

---

## ⚙️ Installation

```bash
# Install via Go
git clone https://github.com/sergiogswv/warden.git
cd warden
go build -o warden main.go
```
